Modern E-Commerce Platform
ShopEase ek feature-rich e-commerce web application hai jo ek real-world shopping experience provide karti hai. Is project mein design aur dynamic functionality ka perfect balance hai.

 Key Features
User Authentication: localStorage based Login/Signup system jo user sessions ko manage karta hai.
Dynamic Cart System: Real-time cart updates jahan users products add, remove aur quantity adjust kar sakte hain.
Smart Product Filtering: Category-based filters aur price/rating ke according sorting functionality.
Integrated Search: Pure catalog mein products instantly search karne ke liye optimized search bar.
Order Checkout: Ek complete flow jo subtotal, 18% GST aur shipping charges calculate karke Order ID generate karta hai.
Fully Responsive: Har device (Mobile, Tablet, Desktop) par seamless chalta hai.

🛠️ Technical Highlights
State Persistence: User ka data aur cart items refresh ke baad bhi save rehte hain (localStorage implementation).
DOM Manipulation: Vanilla JavaScript ka use karke dynamic product cards aur elements render kiye gaye hain.
Advanced CSS: Flexbox, Grid, aur custom animations ka use karke ek modern Dark-Mode UI create kiya hai.

💻 Tech Stack
Frontend: HTML5, CSS3 (Poppins Font, Linear Gradients)

Logic: Vanilla JavaScript (ES6+)

Deployment: Vercel
